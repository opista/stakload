# Scripts

This directory contains project utility scripts.

## Layout

- `igdb/`: IGDB-related tooling
- `igdb/data/`: generated IGDB artefacts (scraped HTML + extracted JSON maps)

## Task Runner

Use [Task](https://taskfile.dev/docs/getting-started) from the repository root:

- `task igdb:extract`
- `task igdb:apply`
- `task igdb:refresh-descriptions`
- `task igdb:probe -- <resource> [options]`

Examples:

- `task igdb:extract -- --input-html scripts/igdb/data/igdb-endpoints.html`
- `task igdb:probe -- games --fields "name,slug" --limit 5`

## Notes

- `scripts/igdb/data/igdb-endpoints.html` is a generated input artefact and is intentionally ignored by formatter checks.
- `scripts/igdb/data/unmatched-fields.json` is generated by `igdb:apply` and should be reviewed after each run.
